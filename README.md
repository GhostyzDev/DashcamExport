# DashcamExport
A Simple Folder Detection + Audio Player. The program works by constantly reading the Output Folder for any new files added to it. When a new file is detected, the audio will play.


Instructions:

1. In the Program.cs:

Ln 10: Update the folder path in the " " (In the programs case, E:\Dashcam\OBS Dashcam) with your Dashcam Output folder.
Ln 13: Update the SoundFilePath to match where you will output the .exe from this program.

2. Save and Build

3. Open the output .exe file.
This will open a Console Window. You can minimize this and it'll still detect.
